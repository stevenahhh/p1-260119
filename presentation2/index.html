데이터 시각화

데이터 시각화의 목적

  • 대량의 데이터 세트 내 패턴 및 추세 식별
  • 분석 결과의 객체성 확보 및 전달력 강화
  • 데이터 이상치(Outlier) 및 분포의 직관적 파악

주요 라이브러리 사양

라이브러리 주요 기능
Pandas 구조화된 데이터 조작 및 통계 분석
Matplotlib 2D 정적 그래프 생성 및 맞춤형 레이아웃 설정
Seaborn 고수준 인터페이스 기반 통계 그래픽 라이브러리

Step 1: 환경 구성

터미널에 아래 명령어를 입력하여 필요한 패키지를 설치하십시오.

pip install pandas matplotlib seaborn openpyxl

Step 2: 데이터 매체 준비

data.csv 파일을 생성하고 아래 데이터를 입력하십시오.

날짜,상품,판매량,매출액
2026-01-01,노트북,5,5000000
2026-01-02,마우스,20,400000
2026-01-03,키보드,15,1200000

Step 3: 데이터 로드 및 분석

import pandas as pd

# 데이터프레임 생성
df = pd.read_csv('data.csv')

# 기초 통계 정보 출력
print(df.info())
print(df.describe())

Step 4: 실시간 수치 시각화

import matplotlib.pyplot as plt
import seaborn as sns

# 시스템 폰트 설정 (Windows)
plt.rcParams['font.family'] = 'Malgun Gothic'

# 상품별 판매량 분석 차트 생성
plt.figure(figsize=(10, 6))
sns.barplot(x='상품', y='판매량', data=df)
plt.title('Product Sales Performance')
plt.show()
